# Minigame — Словесная дуэль

Python-only прототип на FastAPI + Jinja2.

## Основная логика
- Таймер хода игрока: **17 секунд**.
- Если игрок не успел за 17 секунд — он проигрывает матч.
- Бот отвечает через случайный интервал **3-5 секунд**.
- Если бот сразу видит, что на букву нет слов, он проигрывает мгновенно (без ожидания).
- Очки начисляются только игроку (боту очки не начисляются).
- Редкие слова дают повышенный бонус по частотности (rare / ultra-rare) и визуальный combo-эффект.
- Уровни бота: `easy`, `medium`, `hard`.
- Для каждого матча бот получает случайный пул слов:
- В ботовом пуле всегда поддерживается минимум 5 слов на каждую доступную стартовую букву.
  - easy: 400-500
  - medium: 450-750
  - hard: 500-900
- Бот гарантированно отвечает минимум 2 раза, затем получает небольшой растущий шанс проиграть (на hard рост самый медленный).
- Шанс случайного проигрыша настроен под более длинные матчи: easy обычно около 10-20 раундов, hard заметно дольше.
- Стартовая буква матча выбирается случайно.
- Есть история матчей и лидерборд (по максимальному счету игрока).

## Запуск
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python scripts/seed.py
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

- Игра: http://localhost:8000/
- История: http://localhost:8000/history
- Лидерборд: http://localhost:8000/leaderboard
